# Manual QA Testing Project

This repository contains a hands-on manual software testing practice project created as part of self-learning and academic practice.

## Project Overview
The goal of this project is to demonstrate practical knowledge in manual software testing, including writing test cases, executing tests, and reporting bugs in a structured and professional way.

## Scope of Testing
- Functional testing of core features
- Manual test case design and execution
- Bug reporting with clear reproduction steps
- Basic UI and usability testing
