# Login Page - Test Cases

## TC_01 - Login with valid credentials
Steps:
1. Open login page
2. Enter valid email
3. Enter valid password
4. Click Login

Expected Result:
User is logged in successfully

---

## TC_02 - Login with invalid password
Steps:
1. Open login page
2. Enter valid email
3. Enter invalid password
4. Click Login

Expected Result:
Error message is displayed
